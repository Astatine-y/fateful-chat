# Fateful Chat - AI-Powered Bazi Interpretation

![Deploy](https://github.com/Astatine-y/fateful-chat/actions/workflows/deploy.yml/badge.svg)
![CI](https://github.com/Astatine-y/fateful-chat/actions/workflows/ci.yml/badge.svg)

A production-ready Bazi (Chinese astrology) interpretation system with payment integration, built with the latest web technologies.

## 🚀 Quick Deploy

### Railway (Backend)
```bash
npm run deploy -- --backend
```

### Vercel (Frontend)
```bash
npm run deploy -- --frontend
```

### Or Deploy All
```bash
npm run deploy -- --all
```

## Features

- **Free Daily Calculations** - No login required for 1 calculation/day
- **5 Free Credits** - On user registration
- **Google OAuth** - One-click login
- **Apple Sign In** - (requires Apple Developer)
- **Subscription** - Monthly/Yearly plans
- **Lifetime Deal** - One-time purchase
- **Referral Program** - Earn credits
- **PWA** - Install as app
- **10 Languages** - EN/ZH-CN/ZH-TW/JP/KO/ID/VI/ES/FR/TH

## Tech Stack

- **Backend**: Node.js + Express + TypeScript + MongoDB
- **Frontend**: Next.js 14 + React + TypeScript
- **Payments**: Stripe
- **AI**: OpenAI
- **Auth**: JWT + OAuth
- **i18n**: react-i18next

## Quick Start

```bash
# Install
npm install

# Development
npm run dev

# Production
npm run build
npm run start
```
